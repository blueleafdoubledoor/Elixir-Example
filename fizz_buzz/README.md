# FizzBuzz

**Elixir で FizzBuzz を実装**

コマンドラインから実行するアプリ。
下記のコマンドにてビルドを行う。
```
$ mix escript.build
```

実行時に引数としてFizzBuzz の最後の番号を指定する。
